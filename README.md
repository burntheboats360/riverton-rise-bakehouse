# Riverton Rise Bakehouse

Static website for a Riverton, Utah artisan sourdough and cookie bakery.

## Structure

- `index.html` - main site page
- `thanks.html` - Netlify form success page
- `styles.css` - responsive styling and visual details
- `script.js` - product selection helper
- `assets/` - site images
- `netlify.toml` - Netlify publish settings, redirects, and headers

## Deploy

Netlify can publish this repo directly from the repository root. No build command is required.
